#Arquitetura

* As funções relacionadas ao gerenciamento das casas
 do jogo da velha ficarão no módulo **jogovelha.py**.

* O estado de cada casa do jogo será representada por uma 
string: "." para cada vazia; "X" para cada ocupada pelo 1º
jogador; "0" para casa ocupada pelo 2º jogador.

* A função inicializar() retornará uma lista 3X3, onde casa
posição conterá uma string para indicar o estado de uma 
casa do jogo. A função retornará todas as casas inicialmente vazias.
